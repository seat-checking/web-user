## 주요 기능 📦

### ⭐️ 상품 업로드

- 상품을 등록하며 할인율 입력시 할인가 자동변경
- 할인가가 있을 경우 할인가 상품목록 구역에 업로드

### ⭐️ 상세페이지

- 아이템 클릭시 상세페이지로 이동하여 이미지와 상품에 대한 설명이 나오는 상세페이지 화면 렌더링

### ⭐️ 결제기능

- 결제 버튼 클릭시 메인페이지 렌더, 결제된 상품 soldout
